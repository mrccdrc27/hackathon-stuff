# Building Management Agent Summary
**Source:** [[raw/building_management_agent.md]]
**Ingested:** 2026-05-06

## Summary
Details the role of the Management Agent as the real-time interaction layer for project synchronization, utilizing Gemini Multimodal Live API.

## Key Takeaways
- **Real-time Facilitation:** Conducts standups and syncs with low-latency voice.
- **Hybrid Context:** Combines "baked" artifacts (backlogs, digests) with live project signals (Kanban health, GitHub Actions, Calendar).
- **Sprint Management:** Tracks Velocity, Deadlines, and Definition of Done (DoD) while identifying resource and technical bottlenecks.
- **Vertical Flow:** Input from voice, baked context, and real-time signals processed into meeting summaries and task intents.
