# Project: Paul Notes on Hackathon
**Source:** [[raw/paul notes on hackathon.md]]
**Ingested:** 2026-05-05

## Summary
Proposes an application that acts as a Senior Developer and PM to address the shortcomings of "vibe coding" and automate Scrum methodologies.

## Key Takeaways
- **Solution:** AI app that interrogates code decisions and prioritizes tasks (New Feature -> Business Fit -> Scalability -> Security).
- **Features:** IDE extension, GitHub import, Senior Dev Chat, auto-tasking from vulnerabilities, expertise-based task distribution, and Scrum automation.
- **Tech Stack:** Django, PostgreSQL, GitHub OAuth, Twilio, Vercel/Render, Agno framework, and Gemini API.
- **Agent Architecture:** Three agents: Senior Dev (codebase), PM (business/budget), and Scrum (task generation).
