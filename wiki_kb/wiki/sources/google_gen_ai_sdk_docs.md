# Google Gen AI SDK Documentation (Overview)
**Source:** [[raw/Google Gen AI SDK documentation.md]]
**Ingested:** 2026-05-05

## Summary
Comprehensive documentation for the Google GenAI SDK, providing high-level interfaces for interacting with Gemini models, including the Live API, standard generation, and media processing.

## Key Takeaways
- **Unified Interface:** Supports Python and JavaScript for all Gemini API endpoints.
- **Live API Integration:** Simplifies WebSocket handling with async context managers (`aio.live.connect`) and callback-based event systems.
- **Type Safety:** Provides structured types for `Content`, `Part`, `GenerationConfig`, and `LiveConnectConfig`.
- **Media Support:** Built-in utilities for handling base64 encoding and blob creation for audio, image, and video data.
