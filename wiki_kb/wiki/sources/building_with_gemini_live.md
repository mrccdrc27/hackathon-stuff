# Building with Gemini Live (Pipecat)
**Source:** [[raw/Building with Gemini Live.md]]
**Ingested:** 2026-05-05

## Summary
Pipecat is a framework for building production-ready voice AI agents using Google's Gemini Live API. It supports voice, vision, telephony (via Twilio), and tool use, managing media streaming through WebRTC or WebSockets.

## Key Takeaways
- Pipecat handles turn-taking, voice activity detection (VAD), and media transport (Daily WebRTC, Twilio).
- Supports multimodal interactions including video and screenshare processing.
- Offers a CLI (`pc`) for scaffolding and deploying projects to local or Pipecat Cloud environments.
- Provides starter projects for Phone Bots (telephony) and Web Bots (vision/screenshare).
